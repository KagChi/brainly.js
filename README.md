## Installation

To Use Api Install Package First
```bash
npm install @kagchi/brainly.js
```

## Usage
### search Info
```javascript
(async() => {
const data = require("@kagchi/brainly.js")
return data.search("matematika");
})()
```
```javascript
(async() => {
const data = require("@kagchi/brainly.js")
return data.searchV2("matematika");
})()
```
